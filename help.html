The navbar (navigation bar) is a powerful tool. Using it, you can journey across the enitrety of the website. To get to the home page to access the tutorials, press "Home." Using Tutorials

Tutorials will take you step-by-step through learning a new concept. If something doesn't make sense, tell me at brancy280@gmail.com. The large, bold letters explain what the section is about. The smaller text will guide you through the lesson.

If you see any text inside of a black box with an orange rectangle at the bottom, then you know that the text inside is actually code. This code can be copied and pasted into an HTML, CSS or JavaScript file (whichever is indicated), where it can then be saved an run.

Creating an HTML File

Creating a new file is a breeze! Simply open up your computer's search menu (the Windows icon on Windows, and the search icon in the top right on Mac), and type in "Notepad" (on Windows) or "TextEdit" (on Mac). Open up the first result, and get to typing!

To save the file, press "ctrl" (or "command" on a Mac) and "s" together. This will bring up a menu where you can name your file. Make sure that the file name ends in ".html"! Once it's named, hit "Save." You can then open the file from where you saved it to see your work.
