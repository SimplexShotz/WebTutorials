Titles and Headers

A title is essential to any good webpage. The title is what is displayed in the page's tab. It will also be the default name when your page is bookmarked or favorited. Thankfully, it's quite easy to add one; all we need to do is add a title element. This should be inserted inside the head element.

<!DOCTYPE html>
<html>
  <head>
    <title>Put your title here!</title>
  </head>
  <body>
  </body>
</html>
HTML

If you run the above code, the tab will be called "Put your title here!" You can replace the text in the code with whatever you want!

If you ran the code, you may have noticed that the page itself is blank. It's time to fix this by adding some content!

One of the most common types of text is called a header. A header outlines important parts of the page, telling the visitor what each section will be about. There are actually several predetermined header sizes, ranging from <h1> (the largest) to <h6> (the smallest). The number (in the second example, the 6) can be replaced with any value between 1 and 6. Similarly to the HTML, head, body and title tags, the header tag needs to be closed with </h(number)>.

Note that the header element should be placed inside the body. Let's look at an example.

<!DOCTYPE html>
<html>
  <head>
    <title>Title</title>
  </head>
  <body>
    <h1>This header is largest!</h1>
    <h3>This header is large!</h3>
    <h4>This header is small.</h4>
    <h6>This header is smallest.</h6>
  </body>
</html>
HTML
