The Basic HTML Setup

All HTML should begin by declaring the document type. In our case, this would be HTML.

<!DOCTYPE html>
HTML

This simply tells the computer what to expect next from us. Following this, we should add an HTML tag. This encloses everything, and helps to keep your code more organized.

<!DOCTYPE html>
<html>
</html>
HTML

Now, we need to add a head and a body. The head contains all of the setup, while the body contains everything that is displayed on the visitor's screen.

<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
  </body>
</html>
HTML

At this point, you may have noticed that all of the things we have written so far are surrounded by angle brackets or inequality signs (< and >). These tell the browser that what is inside contains HTML, and not some other scripting language. They, along with their contents are referred to as tags. For example, <html> is called an HTML tag.

Another thing you might have noticed is that most of the tags are doubled, the second one beginning with a slash (/). This slash, followed by the tag name, tells the browser where that element ends (you can learn more about elements on W3Schools).

And that's the basic HTML setup! In the next lesson, you will learn about a few important elements, and begin creating a webpage!
